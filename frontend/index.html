1. Direct HTTP (x402)

# Server returns 402 + payment info

# Client pays with USDC on Base
GET /api/tools/ai-chat
X-PAYMENT: <base64-payload>
Content-Type: application/json

{"query": "Hello Kai!"}

        

        

          
2. MCP Protocol

          
POST /mcp
Content-Type: application/json
X-PAYMENT: <base64-payload>

{
  "jsonrpc": "2.0",
  "method": "tools/call",
  "params": {
    "name": "ai_chat",
    "arguments": {
      "query": "Hello from Claude!"
    }
  }
}

        

        

          
3. A2A Protocol

          
POST /a2a
Content-Type: application/json
X-PAYMENT: <base64-payload>
X-Agent-Address: 0xYourAgent
X-Agent-Name: YourBot

{
  "task": "market_data",
  "params": {"token": "ETH"},
  "callback_url": "https://..."
}
